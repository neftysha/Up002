# up (WPF)

Учебное настольное приложение на C# (WPF) для публикации и просмотра объявлений (аналог Avito).

## Функционал
- Главная страница: список объявлений, поиск по ключевым словам, фильтрация, фото/заглушка
- Авторизация по логину/паролю (без шифрования/хэширования)
- Личный кабинет: добавление/редактирование/удаление объявлений
- Статусы: Активно / Завершено
- При завершении: ввод суммы и фиксация прибыли
- Просмотр завершённых объявлений и общей прибыли

## Технологии
- WPF (.NET Framework)
- Entity Framework (EDMX)
- MS SQL Server

## Запуск
1. Открыть решение в Visual Studio
2. Настроить строку подключения в App.config
3. Запустить проект
